# TROFF

## Effect

Debug instruction to stop/disable tracing of MSX-BASIC program execution and return to normal program execution mode. Using the `NEW` instruction will also disable the tracing.

## Syntax

`TROFF`

## Related to

`NEW`, `TRON`

## Compatibility

MSX-BASIC 1.0 or higher
