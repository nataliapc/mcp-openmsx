# LPOS()

## Effect

Returns the current memory address of the print head within the printer buffer.

## Syntax

`LPOS(<Variable>)`

## Parameter

`<Variable>` is a dummy variable.

## Example

```basic
10 LPRINT "MSX Forever!"
20 LPOS(0)
Â 
RUN
12
```

## Compatibility

MSX-BASIC 1.0 or higher
