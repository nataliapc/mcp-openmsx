# END

## Effect

Closes all open files (if any) and ends program execution.

## Syntax

`END`

## Example

```basic
10 PRINT "The end is nigh!"
20 END
30 PRINT "I shall not be executed"

run
The end is nigh!
```

## Related to

`STOP`

## Compatibility

MSX-BASIC 1.0 or higher
