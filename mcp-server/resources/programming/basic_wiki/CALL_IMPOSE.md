# CALL IMPOSE

## Effect

Sets superimposition mode.

## Syntax

`CALL IMPOSE (<Mode>)`

## Parameter

`<Mode>` is a value between 0 and 2 to set video mode.
- 0 for computer screen (internal synchronization).
- 1 for superimpose (composite video).
- 2 for external video.

## Related to

`COLOR`, `SCREEN`, `SET VIDEO`

## Compatibility

P-BASIC
